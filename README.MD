<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
* yarn
  ```sh
  npm install yarn -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/andrew87e/react-team-gen.git
   ```
2. Install yarn packages
   ```sh
   yarn install
   ```
